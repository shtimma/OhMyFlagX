# Oh My FlagX - Тренажер по Географии

Кроссплатформенное мобильное приложение (Android & iOS), созданное на основе Compose Multiplatform. Приложение предназначено для подготовки детей 12–13 лет к олимпиадам по географии.

## Основные возможности

*   **Режим обучения (Study Mode)**: Интерактивные карточки и тесты по 9 темам (Флаги, Столицы, Население и др.).
*   **Блиц-режим (Blitz)**: Соревнование на время. Имитация многопользовательской игры с таймером (10 секунд на вопрос).
*   **Панель родителя (Parent Dashboard)**: Просмотр статистики успеваемости, времени обучения и проблемных зон.
*   **Многоязычность**: Полная поддержка русского и английского языков.

## Технологический стек

*   **Kotlin Multiplatform (KMP)**: Общая бизнес-логика и данные.
*   **Compose Multiplatform**: Единый UI код для Android и iOS.
*   **Material 3**: Современные компоненты дизайна.

## Структура проекта

*   `shared/`: Общий код.
    *   `commonMain/`: Основная логика приложения (`App.kt`), модели данных и переводы.
    *   `androidMain/` & `iosMain/`: Специфичные для платформ настройки (точки входа).
*   `androidApp/`: Модуль для сборки Android-приложения.
*   `iosApp/`: Проект Xcode для сборки iOS-приложения.

## Как запустить

### Android
1. Откройте проект в Android Studio.
2. Выберите конфигурацию `androidApp`.
3. Нажмите **Run**.

### iOS
1. Откройте проект в Android Studio.
2. Выберите конфигурацию `iosApp` (требуется macOS и Xcode).
3. Нажмите **Run**.

## Сравнение с HTML-прототипом

Приложение полностью переносит функционал оригинального HTML-файла:
*   Сохранены все цвета и градиенты.
*   Логика переключения экранов через `Crossfade` (вместо `display: none` в CSS).
*   Реализован таймер в Блиц-режиме через `LaunchedEffect` и `delay`.
